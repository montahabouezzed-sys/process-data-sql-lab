# 📓 Data Analysis & Visualization

This directory contains the Jupyter Notebooks used to analyze the automotive dataset extracted from the SQL database.

### 📊 Key Analyses
- **Exploratory Data Analysis (EDA):** time‑series plots, moving averages, anomaly detection, test‑run comparison, and dashboard layout
- **Visualizations:** Distribution of anomalies (3 Sigma Quality)
