# 📓 Data Analysis & Visualization

This directory contains the Jupyter Notebooks used to analyze the automotive dataset extracted from the SQL database.

### 📊 Key Analyses
- **Exploratory Data Analysis (EDA):** time‑series plots, moving averages, anomaly detection, test‑run comparison, and dashboard layout
- **Visualizations:** Distribution of anomalies (3 Sigma Quality)


### 📈 Visual Highlights
The analysis identified specific patterns in sensor behavior:
- **Temperature Stability:** Observation of cooling system efficiency over time.
- **Voltage Spikes:** Identifying potential electrical anomalies in the vehicle data.

### 🛠️ Technology Stack
- **Python (Pandas/Matplotlib):** For data manipulation and plotting.
- **SQL (DuckDB):** For efficient extraction of time-series data.
